## Build Type Classification Insights

### Classification Criteria
*(How to identify different build types)*

#### Minion Build Indicators
- **Primary Skills**: Summon Bear, Summon Wolf, Summon Skeleton, Summon Wraith, etc.
- **Supporting Skills**: Often specialized versions used by companions (Bear uses specialized Earthquake)
- **Key Stats**: Minion damage, minion health, minion attack speed, minion resistances, player survivability
- **Unique Items**: Items with minion bonuses, companion enhancements, minion skill levels
- **Critical Insight**: Player damage stats (physical damage, crit) do NOT benefit minions in pure minion builds
- **Companion Persistence**: Primalist companions persist through zone changes and rarely die due to high base defense
- **Analysis Date**: 2025-08-29
